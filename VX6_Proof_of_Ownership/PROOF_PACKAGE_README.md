- `VX6_OWNERSHIP_DECLARATION.md`: signed ownership declaration.
- `VX6_git_history.txt`: full Git commit history across refs.
- `VX6_contributors.txt`: contributor summary across refs.
- `VX6_first_commit.txt`: first commit hash, author, email, date, and subject.
